Wild Goose Dashboard - User Wiki

Introduction

The Wild Goose OEE & TPM Dashboard is a comprehensive tool designed to help brewery operators monitor, track, and optimize their production line performance. It combines Overall Equipment Effectiveness (OEE) metrics with Total Productive Maintenance (TPM) practices to ensure maximum efficiency and minimal downtime.

This dashboard is particularly useful for tracking production runs, identifying bottlenecks, and maintaining equipment health across all stations of your canning line.

Getting Started

1. Starting a Session

Before you begin production, you must start a session:

  • Click the "Start Session" button in the header.
  • The session status will change to Active.
  • All timers and data tracking will begin automatically.

2. Ending a Session

At the end of your shift or production run:

  • Click the "End Session" button.
  • All timers will pause, but all data will be saved.
  • The session status will change to Session Ended.
  • You can restart a new session the next day, and all data will accumulate.
💡 Tip: Always end your session properly to ensure no data is lost. The system automatically saves all metrics to your browser's local storage.

Production Control

Starting and Stopping Production

Use the Start and Stop buttons to control the production line:

  • Start: Begins the production timer. The status will change to Running.
  • Stop: Pauses the production timer. The status will change to Stopped.

Tracking Downtime

If the production line stops unexpectedly (e.g., for maintenance, cleaning, or operator breaks), use the "Operator Downtime" button:

  • Click "Operator Downtime" to start tracking downtime.
  • A section will appear to estimate cans lost during downtime.
  • Enter the number of cans lost and click "Add to Out of Spec" to include them in your quality metrics.
  • Click "Cancel" if the downtime was a false alarm or the cans were not actually lost.
  • Click the downtime timer again to stop tracking.

Cleaning Time

Use the "Cleaning Time" button to track time spent on cleaning equipment. This is separate from production downtime and helps you analyze cleaning efficiency.

⚠️ Important: Always stop the production timer before starting downtime or cleaning timers to ensure accurate data.

Run Data Entry (Totalizer)

The Totalizer section allows you to track the number of cans produced during each run. You only need to enter In-Spec and Out of Spec cans—the total is calculated automatically.

How to Use the Totalizer:

  • Enter the number of In-Spec Cans (cans that meet quality standards).
  • Enter the number of Out of Spec cans (cans with defects, low fills, or other issues).
  • Click "Send to Totalizer" to add these values to your accumulated totals.
  • Use "Reset Run Data" to clear the current run's data without saving.

Note: Cans lost during downtime are automatically added to the "Out of Spec (This Run)" field when you stop the downtime timer or save a breakdown record.

OEE Dashboard

OEE (Overall Equipment Effectiveness) is a key performance indicator that measures how effectively your production line is running. It is calculated as:

OEE = Availability × Performance × Quality

OEE Metrics Explained:

  • Availability: The percentage of time the equipment is available for production (vs. downtime).
  • Performance: The speed at which the equipment runs compared to its maximum capacity.
  • Quality: The percentage of good (in-spec) products out of the total produced.
  • Overall OEE: The product of the three metrics above, giving a comprehensive view of efficiency.

The dashboard automatically calculates these metrics based on your production data and downtime records.

TPM - Total Productive Maintenance

TPM (Total Productive Maintenance) is a proactive approach to equipment maintenance that aims to maximize production efficiency by preventing downtime, defects, and accidents.

Recording a Breakdown:

If a breakdown occurs, record it using the "Record Breakdown" button:

  • Select the Station where the breakdown occurred.
  • Enter the Date and Time of the breakdown.
  • Select the Root Cause (e.g., mechanical, electrical, operator error).
  • Enter the Estimated Cans Lost during the breakdown.
  • Add a Description of the issue.
  • Enter the Downtime Duration in minutes.
  • Optionally, add a Solution Applied and any Replaced Parts.
  • Click "Save" to record the breakdown.

Recorded breakdowns will appear in the station's history and will be used to calculate OEE metrics.

Breakdown Causes:

The dashboard tracks the most common causes of breakdowns by station. This helps you identify patterns and prioritize maintenance efforts. The causes include:

  • Mechanical Issues
  • Electrical Issues
  • Seaming Problems
  • Filling Problems
  • Operator Error
  • Other

Runtime Analysis Charts

The dashboard includes two charts to help you visualize your production data:

1. Time Distribution (Bar Chart)

This chart shows the distribution of time across different activities:

  • Production Time: Time spent actively producing cans.
  • Downtime: Time when the line was stopped due to issues.
  • Cleaning Time: Time spent cleaning equipment.
  • Operator Pause: Time when operators took breaks or paused production.

2. Efficiency Breakdown (Pie Chart)

This chart provides a visual breakdown of how your total time was spent, making it easy to see the proportion of productive vs. non-productive time at a glance.

Exporting Data

You can export all your production data to an Excel file for further analysis or reporting:

  • Click the "Export to Excel" button in the header.
  • The file will include multiple sheets:
    • Production Data: Total cans, in-spec cans, out of spec, timers, and OEE metrics.
    • Maintenance Schedule: All scheduled maintenance slots.
    • TPM Data: All recorded breakdowns, including causes, durations, and solutions.
    • Stations Data: OEE metrics for each station.
  • The file will be named Wild_Goose_Production_Report.xlsx.
💡 Tip: Use the exported data to create custom reports or share production metrics with your team.

Production Line Flow

The Production Line Flow diagram provides a visual representation of your canning line, from Depalletizer to Dating. Each station's status is displayed with:

  • OEE Percentage: The Overall Equipment Effectiveness for that station.
  • Status: OK (green), Warning (yellow), or Critical (red).

Click on any station to view its detailed metrics and breakdown history.

Equipment Health

The Equipment Health section provides an overview of the health status of each station in your production line. Each station is displayed with:

  • Status Emoji: 🟢 (Good), 🟡 (Warning), 🔴 (Critical), or ⚪ (No data).
  • OEE Percentage: The current OEE for the station.
  • Last Maintenance: The date of the last recorded maintenance (if available).

Best Practices

  • Start and End Sessions Properly: Always use the session buttons to ensure accurate data tracking.
  • Record All Downtime: Even short downtimes should be recorded to get accurate OEE metrics.
  • Track Cans Lost: Estimating and recording cans lost during downtime helps you understand the true cost of breakdowns.
  • Review Breakdown Causes: Regularly check the breakdown causes to identify recurring issues and address them proactively.
  • Use the Totalizer: Enter run data regularly to keep your production totals up to date.
  • Export Data Weekly: Export your data weekly to create a backup and analyze trends over time.

Troubleshooting

My data isn't saving. What should I do?

The dashboard uses your browser's localStorage to save data. If your data isn't saving:

  • Make sure you're using a modern browser (Chrome, Firefox, Edge, or Safari).
  • Check that your browser's localStorage isn't full (clear some data if needed).
  • Ensure you're not in Private/Incognito Mode, as some browsers disable localStorage in this mode.

The timers aren't working. What's wrong?

If the timers aren't starting or stopping:

  • Make sure you've started a session using the "Start Session" button.
  • Check that you're not trying to start a timer that's already running.
  • Refresh the page and try again.

How do I reset all data?

To reset all data, you can clear your browser's localStorage:

  • In Chrome: Go to Settings > Privacy and Security > Clear Browsing Data and select Local Storage.
  • In Firefox: Go to Options > Privacy & Security > Cookies and Site Data > Clear Data.
  • Alternatively, open the browser's Developer Tools (F12) and run localStorage.clear() in the Console.
⚠️ Warning: Clearing localStorage will permanently delete all your saved data. Make sure to export your data first if you need to keep it.
